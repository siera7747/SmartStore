--실행 방법--

1. 실행 파일을 연다.

2. 스마트스토어 아이디, 비번(네이버) / 발송 메일을 보낼 판매자 이메일 주소와 비번(지메일)을 각 칸에 입력한다.

3. 스마트스토어 자동화 프로그램 시작 버튼을 누른다.

4. 종료하고 싶을 땐 시작 버튼을 누를때 생성되는 콘솔 창에 Ctrl + C를 입력한 후 조금 기다린다.

5. 자동화 프로그램이 종료된 후 하단부 글자가 버튼 클릭 대기로 바뀌면 재시작 준비가 된 것이다.

--주의 사항--

1. 스토어 아이디와 비번은 반드시 네이버로그인 이어야 한다.

2. 판매자 이메일 주소는 반드시 지메일이어야 한다.

3. 종료 시 약간의 시간이 필요하며 그 동안은 앱이 잠시 응답없음으로 표시될 수 있으나 자동화 프로그램이 종료되면 정상으로 돌아오니 강종할 필요는 없다.

--엑셀에 관해서--

1. 파란색 재고 관리 버튼으로 재고를 편하게 추가할 수 있다.
   이미 존재하는 번호로 추가하면 해당 번호의 재고가 새로 입력한 정보로 바뀌니 주의

2. 자동화 시스템으로 발송된 재고는 재고 란이 사용으로 바뀌며 비고 란에 구매자 정보(이름, 이메일, 발송 시각)가 입력된다.

3. 모든 재고가 사용된 경우 비고 옆 칸에 모든 재고가 사용되었다는 문구가 입력되며 바로 옆칸에 해당 재고를 주문하였지만 지연 메일을 보낸 구매자 이름이 입력된다.
   (이 입력 정보로 지연 메일을 중복으로 보내지 않도록 함)

4. 재고를 추가할 땐 비고 칸이나 비고 다음 칸들의 정보는 적절히 제거하거나 변경할 것을 권장.

5. 모든 엑셀 내용은 수동으로 수정 가능하며, 재고 관리 버튼 메뉴로도 변경 가능하다.

--엑셀 자동화 버튼에 관해서--

파란색 재고 관리 버튼으로 재고관리가 가능하다.
다만 이 버튼으로 관리 가능한 것은 비고란에 적힌 정보까지이며, 자동화프로그램에 의해 입력된 비고란 이후의 추가 정보들은 관리가 되지 않으므로 주의할 것.

1. 기본 재고 추가 - 번호, 상품번호 등 오른쪽의 재고 정보 칸을 채워서 추가 버튼을 누르면 된다.
    기존 재고에 동일한 번호가 존재했다면 해당 재고의 내용이 자동으로 업데이트된다.
    비고 란엔 무엇을 적어도 괜찮지만, 자동화 시스템이 실행되면 자동으로 구매자 정보로 바뀌므로 주의

2. 재고 정보 수정 - 추가 시에 바꾸고 싶은 번호를 입력한 후 추가하면 된다.
    또는 검색에 해당 번호를 입력하여 찾은 후 수정할 부분을 바꾼 후 수정 버튼을 클릭한다.

3. 재고 정보 삭제 - 삭제하고 싶은 재고 번호를 검색한 후 삭제 버튼을 누른다.
                         
4. 오름차순 / 내림차순 - 오름차순, 내림차순 버튼을 누르면 자동으로 정렬된다.

5. 닫기 - 닫기 버튼을 누르면 재고 관리 자동화 창이 닫힌다.
